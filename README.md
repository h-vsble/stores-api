## Installing

### Requirements

- MongoDB `v.4.x`

## Running

Copy `.env.example` to `.env` replacing the informations with the correct ones.

Run `npm install`.

Run `npm start` in order to run the app.

## Testing

Run `npm run test` or `npm run test:coverage`.
